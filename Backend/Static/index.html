Welcome to my Recollect API

This API is created using nodeJs , expressJs , BcryptJs , JWT , GoogleAuth and for security purpose i used hashing and salt

this api is using mongoDB atlas for getting and storing data and deployed in heroku, the web app using this api is:

recollect-app

HOW TO USE:

1. Use EndPoint https://myrecollect-app.herokuapp.com/posts to fetch all posts and notes your users have saved as a recollect

2. Use EndPoint https://myrecollect-app.herokuapp.com/users to fetch all users using your app and their name and id but without account details
